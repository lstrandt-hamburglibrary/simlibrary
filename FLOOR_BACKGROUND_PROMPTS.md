## How to Use These Prompts

1. Copy a prompt above (includes the style anchor!)
2. Paste into **Bing Image Creator** (free), **Leonardo.ai** (free tier), or **ChatGPT with DALL-E**
3. If aspect ratio is wrong, add: "Make this a wide horizontal banner, 7:1 aspect ratio"
4. Generate 4 images, pick the one that best matches existing backgrounds
5. Download and rename to `floor-[id].png` (e.g., `floor-mystery.png`)
6. Place in `/assets/` folder
7. The game will automatically use it if the floor ID matches

---

## Tips for Consistency

1. **Always use the full prompt** - Don't skip the style anchor at the end
2. **Generate batches** - Do similar floors together (all fiction, all food, etc.)
3. **Compare to existing** - Open an existing background and compare colors/style
4. **Regenerate if needed** - If one doesn't match, try again with same prompt
5. **Same tool** - Stick to one AI tool for all images if possible
